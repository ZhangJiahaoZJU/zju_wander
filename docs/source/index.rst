Welcome to zju_wander's documentation!
===================================

**zju_wander** (/zju_wander/) is a Python library for simulation the two-dimensional vehicle's behavior
in mixed CAV environment.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
